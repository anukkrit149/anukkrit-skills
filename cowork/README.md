# anukkrit-skills-cowork

Curated 50-skill bundle for Cowork. Excludes dev-framework skills (LangChain, LangGraph, Storybook, Vercel, etc.) that only make sense in a coding environment.

## Install
1. Click the .plugin file in Cowork (or drag to the plugin installer).
2. Restart Cowork.
3. Type `/skills` to verify they loaded.
