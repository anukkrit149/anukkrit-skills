# anukkrit-skills-cloud

Full 105-skill library for Claude Code cloud sandbox. Includes everything: LangChain/LangGraph/LangSmith stack, deep-agents, GitNexus, Storybook, Vercel, React/frontend, testing, data/GPU, prompt engineering, and document tooling.

## Install (via git)
1. Push this folder to a git repo.
2. In the cloud sandbox: `/plugin install <repo-url>`
3. Run `/skills` to verify.
